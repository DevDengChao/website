Serverless Devs Cli

轻便的交互式命令行工具，帮助您快速获得Serverless应用

npm install -g @serverless-devs/s

精简的指令集，极速的使用体验

核心指令只保留 config,init,cli,set,exec 便于记忆，优化执行效率节省200%时间

灵活的无配置指令cli

无配置执行指令可以配合做应用集成测试等能力

开放的源设计方案

打破应用的数据来源单一的问题，通过Registry 您可以用有自己的Serverless Hub
